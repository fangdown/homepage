## 技术决策

1. **使用 Next.js App Router** - 现代 React 全栈框架，便于扩展
2. **使用 Tailwind CSS** - 快速开发，参考站点也使用 Tailwind
3. **静态数据优先** - 先硬编码作品数据，GitHub 集成留作后期
4. **组件化设计** - Navbar, Footer, WorkCard 等独立组件
